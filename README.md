## 搬瓦工后台是什么，有几个？

先说个容易混淆的事。搬瓦工有**两个不同的后台**，功能完全不一样，很多人傻傻分不清楚。

**客户区（Client Area）**：这是搬瓦工的账号主后台，用来管理订单、续费、看账单、开工单联系客服。相当于你在这家店的"会员中心"。

**KiwiVM 控制面板**：这是搬瓦工自研的 VPS 管理面板，每台机器独立一个。在这里能看 IP 地址、SSH 端口、流量使用情况，还能重启、重装系统、切换机房、创建快照。相当于服务器的"驾驶舱"。

两者都要会用，缺一不可。

## 搬瓦工登录链接汇总

搬瓦工官网 bandwagonhost.com 在国内经常被 DNS 污染打不开，所以官方提供了几个镜像站地址，效果完全一样：

- **bwh81.net**（最常用，推荐收藏这个）
- bwh88.net
- bwh89.net

**客户区登录地址**：`bwh81.net/clientarea.php`

登录时输入注册邮箱和密码，点 Login 即可。有时候会遇到 Cloudflare 的浏览器验证，多试一次就过了，正常现象。

**KiwiVM 直接登录地址**：`kiwivm.64clouds.com`

不过 KiwiVM 直接打开需要输入服务器 IP 和 KiwiVM 独立密码，比较麻烦。日常用的话，建议从客户区进去更顺。

## 从客户区登录 KiwiVM 的步骤

这是最稳、最常用的方式。步骤如下：

1. 打开 bwh81.net，右上角点 **Client Area**，输入账号密码登录
2. 登录后点顶部菜单 **Services → My Services**
3. 在 VPS 列表里找到你的机器，点右边的 **KiwiVM Control Panel** 按钮
4. 自动跳转并登录到 KiwiVM，不需要额外输入密码

就这四步。进去之后左边是功能菜单，右边是 VPS 的基本信息面板，包括 IP 地址、SSH 端口、流量使用、资源占用等数据一目了然。

## 进了 KiwiVM 之后，这几个功能最常用

KiwiVM 功能不少，新手刚开始用到的主要是这几个：

- **Main controls**：看 VPS 当前状态，可以执行重启、关机操作
- **Root shell → Interactive**：直接在网页里开一个 SSH 终端，不用 Xshell 就能操作服务器
- **Install new OS**：重装/切换操作系统，支持 Ubuntu、Debian、Rocky Linux 等
- **Root password modification**：修改 SSH 的 root 密码
- **Migration to another DC**：一键迁移到其他机房，CN2 GIA-E 套餐支持 12 个以上机房自由切换
- **Two-factor authentication**：开启两步验证，安全性提升用的

说实话 KiwiVM 的界面不好看，但功能实用，用下来不觉得卡，重装系统大概三五分钟搞定。

## 搬瓦工官网打不开怎么办

这个问题挺常见的。bandwagonhost.com 主域名在国内有时候直接 DNS 污染解析不到，解决方法有两个：

**方法一**：直接用镜像站 bwh81.net，功能完全一样，不用额外操作。

**方法二**：修改 hosts 文件或者换一个 DNS（比如 1.1.1.1），可以解析到主域名。不想折腾就直接用 bwh81.net 省事。

顺便提一下，KiwiVM（kiwivm.64clouds.com）偶尔也会出现 Session Expired 报错，刷新一次或者从客户区重新进一般就解决了。持续打不开的话，清一下浏览器缓存再试。

## 注册账号要注意的一件事

搬瓦工注册时有一个坑点：**注册时要关掉代理/梯子**。

原因是搬瓦工有欺诈检测系统，如果你的注册 IP 和付款 IP 差距太大，或者跑的是明显的代理 IP，系统有概率自动判定为欺诈订单，直接退款取消——钱退回来了，但 VPS 也没了。

个人信息不需要完全真实（可以填假名字），但国家/地区建议如实填写，付款信息需要真实。这个踩过的人不少，注意一下。

## 常见问题 Q&A

**Q：搬瓦工后台的登录地址是什么？**

A：客户区登录地址是 bwh81.net/clientarea.php，输入注册邮箱和密码即可。KiwiVM 直接登录地址是 kiwivm.64clouds.com，但建议从客户区跳转进去更方便。

**Q：搬瓦工官网 bandwagonhost.com 打不开怎么办？**

A：直接用官方镜像站 bwh81.net，功能完全一样，无需任何额外配置。

**Q：KiwiVM 登录提示 Session Expired 是什么问题？**

A：这是 KiwiVM 的 Session 超时或者浏览器缓存问题。解决办法：从客户区重新点击 KiwiVM Control Panel 按钮跳转进去，或者清除浏览器 Cookie 和缓存后重试。

**Q：搬瓦工怎么切换机房？**

A：进入 KiwiVM 控制面板，左侧菜单找到 **Migration to another DC**，选择想去的机房，点击迁移即可。CN2 GIA-E 套餐可以切换的机房最多，包括 DC6、DC9、日本软银、荷兰联通等超过 12 个。迁移过程中 VPS 会短暂中断，数据不会丢失。

**Q：搬瓦工支持支付宝吗？**

A：支持。结账页面选择 Alipay 即可，直接扫码付款，不需要海外信用卡。
